# Harmony of Java Design System

Version: 1.0

---

# Overview

This document defines the visual language, UI rules, component consistency,
design philosophy, spacing system, typography, color system,
interaction model, accessibility rules, and responsive behaviors.

Every page and every component must follow this document.

Never create custom styles that contradict this design system.

---

# Design Philosophy

Harmony of Java combines

- Modern Interface
- Luxury Editorial
- Javanese Culture
- Minimalism
- Storytelling
- Cinematic Experience

The interface should feel

Elegant

Warm

Romantic

Premium

Timeless

Never overcrowded.
